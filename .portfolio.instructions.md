# Portfolio Template - Instructions & Guidelines

## Project Overview
Modern, multilingual developer portfolio built with Next.js featuring dark/light themes, beautiful animations, and comprehensive sections for showcasing work and skills.

## Tech Stack

### Core Framework
- **Next.js 14+** (App Router)
- **TypeScript** for type safety
- **React 18+**

### Styling & UI
- **Tailwind CSS** for utility-first styling
- **CSS Variables** for theming
- **Framer Motion** for animations and transitions
- **Lucide React** for icons

### Internationalization
- **next-intl** for translations
- **Supported Languages**: English (en), French (fr), Arabic (ar)
- RTL support for Arabic

### Theme Management
- **next-themes** for dark/light mode
- Persistent theme selection
- System preference detection

### Features & Integrations
- **React Hook Form** for contact form
- **Resend** or **EmailJS** for email integration
- **React PDF** or simple download for resume
- **Intersection Observer API** for scroll animations

- Support RTL layout for Arabic
- Language switcher in header
- All content strings in translation files

### 2. Theme System
- Dark and light mode support
- Smooth theme transitions
- Theme persistence in localStorage

### 3. Animations & Transitions
- Framer Motion for page transitions

## Styling Guidelines

### Color Scheme
```css
/* Light Mode */
--background: 0 0% 100%;
--foreground: 222.2 84% 4.9%;
--primary: 221.2 83.2% 53.3%;
--accent: 210 40% 96.1%;

/* Dark Mode */
--background: 222.2 84% 4.9%;
--foreground: 210 40% 98%;
--primary: 217.2 91.2% 59.8%;
--accent: 217.2 32.6% 17.5%;
```

### Typography
- Headings: Inter or Poppins (bold)
- Body: Inter or Open Sans
- Code: JetBrains Mono or Fira Code

### Spacing
- Consistent spacing scale (4px base)
- Generous whitespace
- Clear visual hierarchy

### Animations
- Duration: 200-300ms for micro-interactions
- Duration: 500-800ms for page transitions
- Easing: ease-in-out or custom cubic-bezier
- Respect prefers-reduced-motion

## Responsive Design
- Mobile-first approach
- Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)
- Mobile navigation menu
- Touch-friendly interactive elements
- Optimized images for different screen sizes

## Performance Optimization
- Next.js Image component for optimized images
- Lazy loading for sections
- Code splitting
- Minimal JavaScript bundle
- Static generation where possible

## Content Management

### Adding Projects
Edit `src/data/projects.ts`:
```typescript
{
  id: "project-id",
  title: { en: "Project Title", fr: "Titre", ar: "عنوان" },
  description: { en: "Description", fr: "Description", ar: "وصف" },
  image: "/images/projects/project.jpg",
  tags: ["React", "Node.js"],
  category: "fullstack",
  links: {
    demo: "https://demo.com",
    github: "https://github.com/user/repo"
  }
}
```

### Adding Skills
Edit `src/data/skills.ts`:
```typescript
{
  name: "React",
  category: "frontend",
  level: 90,
  icon: "react-icon"
}
```

### Updating Translations
Edit files in `src/messages/`:
- `en.json` for English
- `fr.json` for French
- `ar.json` for Arabic
